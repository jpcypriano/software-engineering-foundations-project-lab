# Aula — Inovação, Propriedade Intelectual e Registro de Ideias

## Objetivo da Aula

Nesta atividade, cada grupo deverá analisar seu projeto sob a perspectiva de:

- inovação
- diferenciação tecnológica
- impacto social e de mercado
- propriedade intelectual
- potencial de registro ou patente

O objetivo é compreender que projetos de software não envolvem apenas implementação, mas também documentação, autoria e proteção intelectual.
